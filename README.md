# Art Makes a Way (AMW)

Aplicación web desarrollada como proyecto final del ciclo de Desarrollo de Aplicaciones Web (DAW).

## Descripción del proyecto

Art Makes a Way (AMW) es una plataforma web orientada a la comunidad artística local, diseñada para facilitar la visibilidad de artistas emergentes, la difusión de obras y la interacción entre usuarios.

El sistema plantea una solución digital para centralizar en una sola plataforma diferentes funcionalidades relacionadas con el entorno artístico y cultural.

## Arquitectura del sistema

El proyecto se organiza en una arquitectura basada en:

- **Frontend**: interfaz de usuario independiente.
- **Backend**: API REST encargada de la lógica y acceso a datos.
- **Base de datos**: contenido de usuarios, perfiles y futuras entidades.
